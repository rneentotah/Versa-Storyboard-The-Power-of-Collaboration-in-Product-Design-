# Versa-Storyboard-The-Power-of-Collaboration-in-Product-Design-
An innovative graduation project that introduces a dual-packaging solution for wet and dry wipes, separated by an insulating layer. This design ensures convenience and hygiene in a single, sleek package

هذا المشروع عبارة عن ستوري بورد (Storyboard) مكون من 66 لوحة مرئية. قمت برسمه وتجسيده لصالح مجموعة من طلاب التصميم لمشروع تخرجهم.
​مهاراتي في هذا المشروع:
​رسم الشخصيات (Character Design): ابتكار وتجسيد شخصيتي "Ver" و "Sa" لتمثيل المناديل الجافة والمبللة.
​رسم الستوري بورد (Storyboarding): تحويل فكرة المنتج إلى تسلسل أحداث مرئي يوضح كيف يمكن حمل العبوة واستخدامها بسهولة في أي مكان.
​فكرة المنتج:
عبوة مبتكرة تجمع بين المناديل المبللة والجافة في مكان واحد، وهي الفكرة التي قمت بتمثيلها بصرياً في هذه اللوحات.
